# コーディングガイドライン
コードの最低限の品質を保つためのガイドラインです。


## 目的
このガイドラインの目的は以下の通りです。

- コードの最低限の品質を担保すること
- コーディングのルールを統一することによって保守性を向上すること
- ドキュメント化することによって教育コストを減らすこと
- コードの品質を向上するための土台にすること
